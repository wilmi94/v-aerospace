# Ad Astra!
## Welcome to the Space Services Satellite Development Site!

This site shall act as a knowledgebase for present and future satellite development projects within the Space Services department - in short SatDev.

SatDev will host the following **topics**:

* General knowledgebase on how space systems (i.e. satellites, ground stations, user devices) are built.
* Relevant Standards (e.g. ISO, ESCSS, etc.)
* Workflows and procedures
* Documenting the whole development cycle of projects

## Commands

* `mkdocs new [dir-name]` - Create a new project.
* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs -h` - Print help message and exit.
